Droidstrap
==========

A Pelican theme using bootstrap and Droid fonts.


Has
---

* Basic content listing - home, tag, category.
* Single post and single page templates.
* A touch of responsiveness.
* Tag pages.
* Category pages.


Has not (yet)
-------------

* Author pages.
* Multi language support.
